# 关于python相对导入的问题

## 问题重现

我有以下文件：

```text
a
│  aa.py
│  __init__.py
│
├─b
│  │  bb.py
│  │  __init__.py
│  │
│  └─d
│        dd.py
│        __init__.py
│
└─c
    cc.py
    __init__.py
```

现我在`dd.py`中写下以下函数：

```python
def TestFuncInDD():
    print("在dd.py中的函数")
```

又在`cc.py`中写以下代码意欲导入此文件：

```python
from ..b import d
d.dd()
```

但是运行之后出现以下错误：

```text
Traceback (most recent call last):
  File "\a\c\cc.py", line 2, in <module>    
    from ..b import d
ImportError: attempted relative import with no known parent package
```

该如何做才能够正确导入`dd.py`？
